Inheritence

Remember, you can style your body element just like any other HTML element, and all your other elements will inherit your body element's styles.

overriding the body using a CLASS

Our pink-text class overrode our body element's CSS declaration!

Our blue-text class overrode our pink-text element's CSS declaration!

It doesn't matter which order the classes are listed in the HTML element class="blue-text pink-text" or class="pink-text blue-text"

However, the order of the class declarations in the "style" section is what is important. The second declaration will always take precedence over the first.

overriding all CLASS using the ID

It doesn't matter whether you declare the ID attribute CSS above or below class,
since the id attribute will always take precedence.

Override Class and ID Declarations with Inline Styles(style="")

inline override the CLASS and ID

and it doesnt matter the inline order inside the html element

Override All Other Styles by using Important

the keyword !important make it impossible to be overriden
here is the syntax :::
.violet-text {color: violet !important; }
